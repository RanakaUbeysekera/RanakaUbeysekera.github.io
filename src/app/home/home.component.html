Third party information security posture analysis (TPISP) Tool

This tool is to be used to determine compliance of your organization to the third-party information security risk management framework. the results of this analysis depicted as an overall value and compliance to each specific sections can be officially used to compile the framework profile of the organization, as mentioned within the framework.
